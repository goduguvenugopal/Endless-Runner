# Endless Runner

A professional, high-performance Endless Runner web game built with vanilla JavaScript and HTML5 Canvas. Experience fluid animations, responsive design, and a competitive leaderboard system.

## 🌟 Features

-   **Dynamic Cartoon Character:** A highly detailed runner with jointed limbs (knees and elbows), expressive eyes, and smooth running animations.
-   **Responsive Design:** Fully adapts to any screen size, from large desktop monitors to mobile phones (100% viewport utilization on mobile).
-   **Cross-Platform Controls:** 
    -   **Desktop:** Spacebar, Up Arrow, or Mouse Click to jump.
    -   **Mobile:** Intuitive touch-to-jump mechanics.
-   **Leaderboard System:** LocalStorage integration saves your top 5 high scores permanently in your browser.
-   **Difficulty Scaling:** The game speed incrementally increases as your score grows, providing a progressive challenge.
-   **Professional UI:** Modern dark-themed interface with glassmorphism effects, a functional navbar, and polished modals.
-   **Physics-Based Gameplay:** Realistic gravity and jumping mechanics for a satisfying game feel.

## 🚀 Getting Started

No installation or build process is required! 

1.  Download or clone this repository.
2.  Open `index.html` in any modern web browser (Chrome, Firefox, Safari, Edge).
3.  Click or Tap to start your run!

## 🕹️ How to Play

1.  **Start:** Click the screen or press the 'Space' key to begin.
2.  **Jump:** Leap over **Cacti (🌵)** and **Rocks (🪨)** to stay alive.
3.  **Score:** Points are earned for every obstacle successfully cleared.
4.  **Rankings:** Click the **🏆 RANKINGS** button in the navbar to view your all-time high scores.
5.  **Restart:** If you hit an obstacle, tap the **RUN AGAIN** button to try and beat your record.

## 🛠️ Technical Stack

-   **HTML5 Canvas:** For high-performance 2D rendering and procedural character drawing.
-   **CSS3:** Utilizes Flexbox, CSS Variables, Media Queries for responsiveness, and Backdrop-filters for modern UI effects.
-   **Vanilla JavaScript:** Core game loop, physics engine, collision detection, and LocalStorage state management (No external libraries required).

## 📂 Project Structure

-   `index.html`: Main game structure and UI elements.
-   `style.css`: Responsive layouts and professional aesthetics.
-   `script.js`: Core game logic, animations, and data persistence.
