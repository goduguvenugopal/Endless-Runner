# Vexka Runner

A professional, high-performance, neon-themed endless runner web game built with vanilla JavaScript and HTML5 Canvas. Experience fluid animations, responsive design, and a competitive leaderboard system—now fully installable as a Progressive Web App (PWA).

## 🌟 Features

-   **PWA Ready:** Install the game as a standalone app on your home screen.
-   **Offline Play:** Thanks to Service Worker integration, you can play the game anywhere, even without an internet connection.
-   **Dynamic Neon Character:** A highly detailed runner with jointed limbs, expressive eyes, and smooth animations.
-   **Responsive Design:** Fully adapts to any screen size, from ultra-wide monitors to mobile phones.
-   **Cross-Platform Controls:** 
    -   **Desktop:** Spacebar, Up Arrow, or Mouse Click to jump. Press 'P' to pause.
    -   **Mobile:** Intuitive touch-to-jump mechanics.
-   **Leaderboard System:** LocalStorage integration saves your top 5 high scores locally.
-   **Difficulty Scaling:** Game speed incrementally increases as your score grows.
-   **Professional UI:** Modern dark-themed interface with glassmorphism effects and polished modals.
-   **SEO & Social Optimized:** Full Meta, OpenGraph, and Twitter Card support for beautiful sharing previews.

## 🚀 Getting Started

Vexka Runner is built with zero dependencies.

1.  Clone this repository.
2.  Open `index.html` in any modern web browser.
3.  (Optional) Click the "Install" icon in your browser's address bar to add it to your device as an app.

## 🕹️ How to Play

1.  **Start:** Click the screen or press 'Space' to begin.
2.  **Jump:** Leap over obstacles to stay alive.
3.  **Score:** Points are earned for every obstacle cleared.
4.  **Rankings:** Check the **🏆 RANKINGS** button for your high scores.
5.  **Pause:** Use the **PAUSE** button or press 'P' to take a break.

## 🛠️ Technical Stack

-   **HTML5 Canvas:** High-performance 2D rendering for the game loop.
-   **Vanilla JavaScript:** Core physics engine, collision detection, and state management.
-   **CSS3:** Flexbox, Media Queries, and Backdrop-filters for a modern UI.
-   **Service Workers:** Enables offline asset caching and PWA functionality.
-   **Web App Manifest:** Defines the app's look and feel on mobile home screens.

## 📂 Project Structure

-   `index.html`: Main game structure, PWA registration, and Meta tags.
-   `style.css`: Layout, animations, and neon styling.
-   `script.js`: Core game logic and character rendering.
-   `sw.js`: Service worker for offline support.
-   `manifest.json`: PWA configuration.
-   `assets/`: High-quality sound effects and branding logo.
